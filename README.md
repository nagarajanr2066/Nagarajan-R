# NagarajanR


## Problems

1. **Problem-1**: A simple calculator class that performs addition, subtraction, multiplication, and division.
2. **Problem-2**: Generate a series of odd numbers up to the input number.
3. **Problem-3**: Generate a series of odd numbers up to the input number, but with different logic.
4. **Problem-4**: Count how many numbers in a list are multiples of each number from 1 to 9.
